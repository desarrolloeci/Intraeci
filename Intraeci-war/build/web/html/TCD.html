TCD

Este taller busca conocer las opiniones de la Consejo Directivo sobre diferentes aspectos de la vida universitaria.. En cada grupo habrá un moderador que formulará las preguntas, coordinará la discusión y obtendrá el consenso antes de registrar en la planilla una sola respuesta para cada pregunta. Dicho consenso irá acompañado de una justificación grupal para cada una de las respuestas. Agradecemos su participación
